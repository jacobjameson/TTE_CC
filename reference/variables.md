# Data conventions & toy-data codebook

## Long / person-time format
All analyses use **long (person-time)** data: one row per person per time interval (here, per week).
A minimal row carries:
- **id** — person identifier (time-fixed).
- **time** (or **cal_time**) — interval index since start of follow-up; `timesqr` = its square.
- **covariates Lₖ** — value at the *start* of interval k (time-fixed ones repeat; time-varying ones change).
- **treatment Aₖ** — value at the *end* of interval k.
- **outcome Yₖ₊₁** — indicator that the event occurred in interval k+1.

Time-fixed variables (e.g. baseline age, randomization arm) repeat on every row for a person;
time-varying variables (e.g. symptoms, doses received) may differ across a person's rows.

## Toy-data codebook
Generated by `data/make_toy_data.R` (synthetic; original to this repo). Two datasets:

### `vac_toy_random` — randomized trial
| Variable | Type | Meaning |
|---|---|---|
| id | int | person id |
| time, timesqr | int | follow-up week (0–23) and its square |
| age, age_cat | num/int | age in years; 14-level age category |
| sex | 0/1 | 0 male, 1 female |
| race | 1/2/3 | White / Black / Other |
| urban | 0/1 | urban residence |
| baseline_risk | 1–5 | # severe-disease risk factors (1 = 0 RF … 5 = 4+) |
| smoke | 1/2/3 | never / former / current |
| bmi, obese | num/0–1 | BMI; obese = BMI ≥ 30 |
| diabetes, heartd, ckd, liverd, cancer | 0/1 | comorbidities |
| pcp_visits, flu_vac | 1–4 | healthcare-utilization categories |
| random | 0/1 | randomization arm (0 placebo/none, 1 VACX) |
| treat | 0/1 | received treatment (= `random` at baseline) |
| hosp | 0/1 | hospitalization in this interval (primary outcome) |
| titer | num | antibody titer, U/mL (secondary, continuous) |

### `vac_toy_obs` — observational study
All of the above baseline covariates, plus:
| Variable | Type | Meaning |
|---|---|---|
| cal_time, cal_timesqr | int | calendar week since baseline and its square |
| elig_1 | 0/1 | meets eligibility criteria (used as the broad eligibility flag) |
| elig_2 | 0/1 | eligible **and** initiates first dose (for cloning analyses, v2) |
| immuno | 0/1 | immunosuppressive condition |
| pregnant | 0/1 | pregnant |
| treat, treat_b | 0/1 | (baseline) treatment; confounded by covariates |
| hosp | 0/1 | hospitalization in this interval |
| death | 0/1 | death in this interval (competing event) |
| censor | 0/1 | loss to follow-up in this interval |

### Known truth (`data/toy_truth.rds`)
True marginal 24-week risks under "all treated" vs "all untreated" (g-formula on the true hazard),
for both datasets: `risk1`, `risk0`, `rd`, `rr`. Used by examples and the test suite.

**Built-in lessons:** treatment in `vac_toy_obs` is assigned more often to older / higher-risk /
more-health-seeking people (confounding), and loss to follow-up is informative — so a naïve analysis
is biased while matching / standardization / IP weighting recover the truth.
